# 🧠 Enhancing Image Classification with CNN, RNN and ResNet

A deep learning project that implements and compares three neural network architectures — CNN, RNN and ResNet — from scratch using PyTorch on the MNIST dataset.

---

## 🧪 Techniques Used
- Convolutional Neural Networks (CNN) — 87% accuracy
- Recurrent Neural Networks (RNN) — 90% accuracy
- Residual Networks (ResNet) — 94% accuracy
- CrossEntropyLoss
- SGD & Adam Optimizers
- Loss & Accuracy visualization across epochs
- Kaggle competition submission

---

## 🛠️ Libraries Used
| Category | Tools |
|---|---|
| Language | Python 3 |
| Deep Learning | PyTorch, torch.nn |
| Data Loading | torchvision |
| Visualization | Matplotlib |
| Environment | Google Colab |

---

## 📊 Results
| Model | Accuracy |
|---|---|
| CNN | ~87% |
| RNN | ~90% |
| ResNet | ~94% |

## 🚀 How to Run
```bash
pip install torch torchvision matplotlib numpy pandas
```
Open the `.ipynb` file in Jupyter Notebook or Google Colab and run cells sequentially.

> ⚠️ Update the dataset paths in the notebook to match your local setup before running.
